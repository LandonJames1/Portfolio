Now incorporating the brands dataset. Performing a K-means clustering exercise. Using all twelve brand ratings provided (complex, cool, daring, exciting, intelligent, leader, reliable, technical, trendy, unique, visible, young) in the K-means clustering. Using the elbow method to argue for a particular number of clusters as the right number of clusters

brand_ratings <- brands[, c("complex", "cool", "daring", "exciting", "intelligent", 
                             "leader", "reliable", "technical", "trendy", 
                             "unique", "visible", "young")]

wcss <- sapply(1:10, function(k) {
  kmeans_model <- kmeans(brand_ratings, centers = k)
  kmeans_model$tot.withinss
})

plot(1:10, wcss, type = "o", xlab = "# of Clusters", ylab = "WCSS", main = "Optimal # of Clusters")
